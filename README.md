# Prediction of Supply Chain Health Using Satellite Imagery

## 📌 Project Overview
This repository contains the codebase and research report for predicting global supply chain health and tracking port congestion using satellite imagery and deep learning. 

## 📂 Repository Structure
- `satellite_supply_chain_model.ipynb`: The main Jupyter Notebook containing the data preprocessing, model architecture, training loop, and evaluation metrics.
- `ship_detector_fixed.keras`: The saved pre-trained Keras model for direct inference. Link to download: https://drive.google.com/file/d/1UEX9-FNIaJNLMMjJHE4Qc5Phz9FhO0ED/view?usp=drive_link
- `Supply_Chain_Satellite_Research_Report.pdf`: The comprehensive research paper detailing the methodology, algorithms, and analytical findings.

## 🛠️ Tech Stack
- **Languages:** Python
- **Libraries:** TensorFlow/Keras, Pandas, NumPy, Scikit-learn

## 🚀 How to Use
1. Clone this repository to your local machine.
2. Open `satellite_supply_chain_model.ipynb` to view the pipeline. 
3. **⚠️ Important Note for Reviewers:** To save time and avoid retraining the model from scratch, you can **skip the model training cell**. Instead, simply run the cell that loads the pre-trained `ship_detector_fixed.keras` file to immediately execute the evaluation and inference steps.
