# Prediction of Supply Chain Health Using Satellite Imagery

## 📌 Project Overview
This repository contains the codebase and research report for predicting global supply chain health and tracking port congestion using satellite imagery and deep learning. 

## 📂 Repository Structure
- `satellite_supply_chain_model.ipynb`: The main Jupyter Notebook containing the data preprocessing, model architecture, training loop, and evaluation metrics.
- `port_congestion_model.keras`: The saved pre-trained Keras model for direct inference. *(Note: If file is too large and on GDrive, say: Link to download the weights is [here])*
- `Supply_Chain_Satellite_Research_Report.pdf`: The comprehensive research paper detailing the methodology, algorithms, and analytical findings.

## 🛠️ Tech Stack
- **Languages:** Python
- **Libraries:** TensorFlow/Keras, Pandas, NumPy, Scikit-learn, [thêm các thư viện xử lý ảnh vệ tinh nếu có, ví dụ: OpenCV, Rasterio]

## 🚀 How to Use
1. Clone this repository to your local machine.
2. Ensure all dependencies are installed via `pip install -r requirements.txt` (or install the libraries mentioned in the Tech Stack).
3. Open `satellite_supply_chain_model.ipynb` to view the pipeline. 
4. To run inference without retraining, load the `port_congestion_model.keras` directly into the notebook.
